# **Projekt : Strzelnica**
**Inżynieria procesów biznesowych i systemów regułowych II**

Celem jest zbudowanie modelu strzelnicy, oraz opisaniu procesów jakie dla niej zachodzą i stworzeniu dla nich odpowiednich diagramów.

# **Procesy**
- Rejestracja i weryfikacja klienta
- Rezerwacja
- Przydział sprzętu i stanowiska
- Sesja strzelecka
- Anulowanie / zmiana rezerwacji
